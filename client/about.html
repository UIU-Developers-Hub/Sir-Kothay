About Sir Kothay

"Sir Kothay Achen" is a lightweight web application designed to make sharing messages simple and efficient. Whether you want to leave a quick note, share a thought, or send a message to someone, Sir Kothay gives you a unique, shareable URL and QR code for easy access.

Fast

How it works

From note to delivery in 4 steps

Step 1

Create a Message

Write your message on the home page and save it.

Step 2

Generate a URL

Each message gets a dedicated page with a unique URL.

Step 3

Share via QR

A QR code is automatically generated for quick sharing.

Step 4

View Instantly

Anyone with the link or QR code can open your message.

Tech Stack

Backend: Django (Python)

Frontend: Custom CSS Design System

Database: SQLite (upgradeable)

Hosting: PythonAnywhere

Additional: QR Code Generation

Core Features

  • Message Creation: Write and save your message.
  • Unique URL: Each message gets a unique link.
  • QR Code: Share messages easily with a QR code.
  • Message Viewing: Anyone with the link can view it.
  • User Profiles: Register and manage your messages.

Future Enhancements

Scheduling Messages: Set messages to appear at specific times.
Messaging System: Send direct messages to other users.
Dashboard: Manage all your messages in one place.
